Fetch Vulnerable Components

Use this API to fetch SCA vulnerable components

 GET     https://hacksaw.zohocorpcloud.in/api/vulnerablecomponentsreport


Scope

User OAuth - Hacksaw.REPORT.READ


Throttle:

Limit - 50 requests

Duration - 1 min

Lock period - 5 mins

Throttle type - IP
